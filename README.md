# Unfolding

Repository for the source code and input files for the topological defects based unfolding project. 

The project is currently published at the "International Journal of Solids & Structures" - https://www.sciencedirect.com/science/article/abs/pii/S0020768326000211

An enhancement to the unfolding path search in this work has been applied using reinforcement learning (Maskable PPO) and the repository for that work can be found here - "https://github.com/CBL-UCF/RL_unfolding_path"

Please follow the README.txt file in 'SourceCode' directory and 'InputFiles' directory for explanation about how to use the code.
